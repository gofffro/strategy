# strategy
Есть интерфейс IStrategy с методом операции.

Несколько конкретных стратегий (сложение, вычитание, умножение).

Класс Context содержит текущую стратегию и использует её для вычислений.

В Main меняем стратегию и выполняем операции.

Паттерн Strategy позволяет менять алгоритмы во время выполнения программы.1
